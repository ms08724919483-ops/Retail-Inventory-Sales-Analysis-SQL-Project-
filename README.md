### 4. Low stock alerts -

**SQL Code**
```sql
select p.product_id, p.product_name, sum(i.stock_quantity) as Total_stock,
```

### 10. Inventory coverage (how many days stock will last at current sales rate) -

**SQL Code**
```sql
select p.product_name, 
		sum(i.stock_quantity) as Total_stock,
        round(sum(i.stock_quantity) / (sum(s.quantity) / count(distinct s.sale_date)), 1) as days_coverage
from sales as s
join products as p on p.product_id = s.product_id
join inventory as i on i.product_id = s.product_id
group by p.product_name
order by days_coverage asc;
```
